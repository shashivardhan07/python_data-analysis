# AI Financial & Market Data Analysis 

## Overview
This repository contains a data analysis project exploring the financial performance and market trends of three major AI industry leaders: Google, Meta, and OpenAI. The analysis is performed in a Jupyter Notebook using a realistic synthetic daily dataset spanning from January 1, 2015, to December 31, 2024. It provides visual and statistical insights into how research and development (R&D) spending, product launches, and general market momentum impact AI revenue and stock prices.

## Dataset Information
The analysis utilizes the `ai_financial_market_daily_realistic_synthetic.csv` dataset. The dataset contains 10,959 rows and tracks the following key metrics:
* **Date**: Daily records from 2015 to 2024.
* **Company**: Google, Meta, or OpenAI.
* **R&D_Spending_USD_Mn**: Daily R&D expenditure in millions of USD.
* **AI_Revenue_USD_Mn**: Daily AI-driven revenue in millions of USD.
* **AI_Revenue_Growth_%**: Percentage of AI revenue growth.
* **Event**: Significant company events, such as product releases or policy updates.
* **Stock_Impact_%**: The percentage change in the company's stock value.
* **Year**: A derived column extracted from the Date for annual aggregation.

## Key Explorations & Visualizations
The notebook covers several core analytical themes:
* **Overall Spending & Revenue**: Bar charts comparing the total amount spent on R&D versus the total AI revenue generated by Google, Meta, and OpenAI over the 10-year period.
* **Stock Volatility & Trends**: Line charts tracking the daily fluctuations in stock value (`Stock_Impact_%`) for each individual company.
* **Event Impact Analysis**: Identifies which specific days and events triggered the highest stock impact for each company. Deep dives are included for major events like Google's TensorFlow open-source release and OpenAI's GPT-4 release.
* **Revenue Growth Trajectories**: Scatter and line plots detailing the AI revenue growth percentages year-by-year, illustrating the rapid acceleration in AI monetization.
* **Year-Over-Year Macro Trends**: A combined line chart showcasing the aggregate R&D expenditure versus revenue year-by-year across the industry.
* **Variable Correlation**: A heatmap and a seaborn pairplot displaying the correlations between numeric variables like R&D spending, revenue, and stock impact.

## Technologies Used
The analysis is built using the following Python libraries:
* `pandas`: For data manipulation and aggregation.
* `matplotlib.pyplot`: For creating line charts, bar plots, and subplots[cite: 2].
* `seaborn`: For advanced statistical visualizations like scatterplots, pairplots, and heatmaps.

## How to Run
1. Clone this repository to your local machine.
2. Ensure you have Python installed along with the required libraries (`pandas`, `matplotlib`, `seaborn`).
3. Place the `ai_financial_market_daily_realistic_synthetic.csv` file in the same directory as the notebook.
4. Open the Jupyter Notebook and run the cells sequentially to reproduce the analysis and visualizations.
