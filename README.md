# [cite_start]AI Financial & Market Data Analysis [cite: 3]

## Overview
[cite_start]This repository contains a data analysis project exploring the financial performance and market trends of three major AI industry leaders: Google, Meta, and OpenAI[cite: 10]. [cite_start]The analysis is performed in a Jupyter Notebook using a realistic synthetic daily dataset spanning from January 1, 2015, to December 31, 2024[cite: 2]. [cite_start]It provides visual and statistical insights into how research and development (R&D) spending, product launches, and general market momentum impact AI revenue and stock prices[cite: 8, 88].

## Dataset Information
[cite_start]The analysis utilizes the `ai_financial_market_daily_realistic_synthetic.csv` dataset[cite: 2]. [cite_start]The dataset contains 10,959 rows and tracks the following key metrics[cite: 2, 8]:
* [cite_start]**Date**: Daily records from 2015 to 2024[cite: 2].
* [cite_start]**Company**: Google, Meta, or OpenAI[cite: 10].
* [cite_start]**R&D_Spending_USD_Mn**: Daily R&D expenditure in millions of USD[cite: 8].
* **AI_Revenue_USD_Mn**: Daily AI-driven revenue in millions of USD[cite: 8].
* **AI_Revenue_Growth_%**: Percentage of AI revenue growth[cite: 8].
* **Event**: Significant company events, such as product releases or policy updates[cite: 8].
* [cite_start]**Stock_Impact_%**: The percentage change in the company's stock value[cite: 8].
* [cite_start]**Year**: A derived column extracted from the Date for annual aggregation[cite: 16, 17].

## Key Explorations & Visualizations
The notebook covers several core analytical themes:
* **Overall Spending & Revenue**: Bar charts comparing the total amount spent on R&D versus the total AI revenue generated by Google, Meta, and OpenAI over the 10-year period[cite: 88, 161].
* [cite_start]**Stock Volatility & Trends**: Line charts tracking the daily fluctuations in stock value (`Stock_Impact_%`) for each individual company[cite: 260, 343].
* [cite_start]**Event Impact Analysis**: Identifies which specific days and events triggered the highest stock impact for each company[cite: 441, 448]. [cite_start]Deep dives are included for major events like Google's TensorFlow open-source release and OpenAI's GPT-4 release[cite: 925].
* [cite_start]**Revenue Growth Trajectories**: Scatter and line plots detailing the AI revenue growth percentages year-by-year, illustrating the rapid acceleration in AI monetization[cite: 511, 542].
* **Year-Over-Year Macro Trends**: A combined line chart showcasing the aggregate R&D expenditure versus revenue year-by-year across the industry[cite: 762, 763].
* [cite_start]**Variable Correlation**: A heatmap and a seaborn pairplot displaying the correlations between numeric variables like R&D spending, revenue, and stock impact[cite: 597, 792].

## Technologies Used
The analysis is built using the following Python libraries:
* [cite_start]`pandas`: For data manipulation and aggregation[cite: 2].
* `matplotlib.pyplot`: For creating line charts, bar plots, and subplots[cite: 2].
* [cite_start]`seaborn`: For advanced statistical visualizations like scatterplots, pairplots, and heatmaps[cite: 2].

## How to Run
1. Clone this repository to your local machine.
2. Ensure you have Python installed along with the required libraries (`pandas`, `matplotlib`, `seaborn`).
3. [cite_start]Place the `ai_financial_market_daily_realistic_synthetic.csv` file in the same directory as the notebook[cite: 2].
4. Open the Jupyter Notebook and run the cells sequentially to reproduce the analysis and visualizations.
